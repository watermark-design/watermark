# `@watermark-design/react`

Works with React.

## Installing

```bash
# or pnpm or yarn
npm install @watermark-design/react
```

## Usage

### General

```ts
import { Watermark } from '@watermark-design/react';
```

```jsx
<Watermark width={200} height={200} content={inputValue}>
  <div style={{ height: 400 }}></div>
</Watermark>
```
